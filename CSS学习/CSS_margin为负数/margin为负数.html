margin 为负数，分  上  下  左  右
        1.  margin-left:    -50px;    
        2.  margin-right:   -50px;
        3.  margin-top:     -50px;
        4.  margin-bottom:  -50px;
        （ 因为  行内元素的 特殊， 行内元素  设置  上下padding  上下margin 无效 ）

child div margin-left: 0px
child div margin-left: -20px;   height: 20px;
child div margin-right: -20px;   height: 20px; 
        我认为，在一个元素上设置 margin 为负数（首先设置 margin 要有效）
        就是 告知 所有元素 在某个方向上还有 多少位置  可以去占据
        例如： 上面的例子  
            告知 自身元素  在 child_div 左边还有  20px 的位置 你可以去占据。
            又因为 child_div 块级元素 占据一行（如果该元素没有设置 宽度）
            所以， child_div 就去占据了，没有丝毫在意 它占据的是什么
            -------------
            如果  child_div  设置了一定的宽度，因为 块级元素的  占据整行，
            且 是从 父级元素的  content 区域左边界开始
            因为 告知 child_div 左边有富于位置，故 child_div 会向左移动一点位置，然后开始排布
